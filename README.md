# E-Commerce Customer Churn Analysis

## Overview
This project analyzes customer churn in an e-commerce dataset to identify patterns and factors contributing to customer attrition.

## Tools Used
- SQL
- Python
- Excel
- Power BI

## Key Tasks
- Data cleaning and handling missing values
- Outlier detection and removal
- Data transformation and feature engineering
- Churn analysis and customer segmentation
- Business insights and visualization

## Outcome
The analysis provides insights that help businesses reduce churn and improve customer retention strategies.
